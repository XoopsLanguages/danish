BESKRIVELSE

Profilmodulet er til styring af brugerdefinerede brugerprofilfelter.

INSTALLER/AFINSTALLER

Ingen særlige foranstaltninger nødvendige, følg standardinstallationsprocessen – udpak /profile-mappen i ../modules-mappen. Installer modulet gennem Admin -> Systemmodul -> Moduler.

Detaljerede instruktioner om installation af moduler er tilgængelige i XOOPS betjeningsvejledning

BETJENINGSVEJLEDNING

For at opsætte dette modul skal du:

  • Konfigurer dine præferencer for modulet (se 'Præferencer') og eventuelt profilblokken, hvis du har til hensigt at bruge det (se 'Blokkere').
  • Rediger eksisterende kategorier eller tilføj nye.
  • Rediger eksisterende felter, eller tilføj nye. Her kan du angive, hvilke felter der skal være synlige i hvilken kategori, og om de vil være synlige under brugerregistrering.
  • Definer rækkefølgen af registreringstrin.
  • Og endelig kan du indstille tilladelser for individuelle felter - hvilke der er redigerbare, hvilke er søgbare.

Anti-Spam-foranstaltninger

For at minimere spamregistreringer skal du gøre følgende:

  • gå til Protector-modulet i Admin, gå til Præferencer og derefter nederst ved denne mulighed: "Stop Forum Spam" Kontrollerer POST-data mod spammere, der er registreret på www.stopforumspam.com-databasen.
    Indstil den til "Forbyd IP (ingen grænse)"

  • i /class/captcha/config.php, sørg for at tilstanden er indstillet som "tekst":

    returner $config = array(
    'disabled' => falsk, // Deaktiver CAPTCHA
    'mode' => 'tekst', // standardtilstand, du kan vælge 'tekst', 'billede', 'recaptcha' (kræver api-nøgle)
    'name' => 'xoopscaptcha', // captcha navn
    'skipmember' => sand, // Spring over CAPTCHA-tjek for medlemmer
    'maxattempts' => 10, // Maksimalt antal forsøg for hver session
    );

  • I profilmodulet skal du gå til Admin og i det grundlæggende trin på:
    /modules/profile/admin/step.php?id=1
    sæt "Gem efter stop" til "Nej"


  • Indstil "Brug Captcha efter det andet registreringstrin" i Profilindstillinger til "Ja" (det er standard)
