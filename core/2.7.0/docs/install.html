Installation af XOOPS

Førstegangsinstallation

Forord:

XOOPS er en open source Objektorienteret webpubliceringssystem skrevet i PHP. Det er et ideelt værktøj til udvikling af små til store dynamiske community-websteder, interne virksomhedsportaler, virksomhedsportaler, weblogs og meget mere.

XOOPS frigives i henhold til vilkårene i GNU General Public License (GPL) version 2 eller nyere, og er gratis at bruge og ændre. Det er gratis at videredistribuere, så længe du overholder distributionsbetingelserne i GPL.

Krav

  • WWW Server (Apache, NGINX, IIS osv.)
  • PHP 8.2 eller højere, 8.4+ anbefales
  • MySQL 5.7 .8 eller højere, MySQL 8.4.5+ anbefales

Før du installerer

  1. Konfigurer WWW-server, PHP og databaseserver korrekt.
  2. Forbered en database til din XOOPS-side.
  3. Forbered brugerkonto og giv brugeren adgang til databasen.
  4. Gør mapperne og filerne skrivbare: %s
  5. Af sikkerhedshensyn anbefales det kraftigt at flytte de to nedenstående mapper ud af dokumentrod og skift mappenavnene: %s
  6. Opret (hvis ikke allerede til stede) og gør mapperne skrivbare: %s
  7. Slå din browsers cookie og JavaScript til.

Installerer lokalt

Hvis du kører et lokalt miljø til udvikling eller test, så sørg for at have det tidligere krav opfyldt. Når dette er gjort, skal du kopiere indholdet af htdocs bibliotek (fra XOOPS distributionsfil) til roddokumentstien til dit webmiljø. Når filerne er kopieret dertil, du kan starte installationen ved at pege din browser til "http://dit websted.com". Dette starter installationsprocessen.

Installation på en hostet platform

Hvis du kører i et hostet miljø, skal du pakke XOOPS-filerne ud lokalt eller på serveren, hvis du har shell (dvs. SSH) adgang. Når det er gjort, skal du sørge for at flytte eller kopiere alle XOOPS-filer fra htdocs bibliotek til din rodwebmappe (din udbyder giver normalt denne placering med retninger). Når filerne er kopieret dertil, kan du starte installationen ved at pege på din browser "http://dit websted.com". Dette starter installationsprocessen.
